# Create a message template

This guide walks you through creating an SMS message template in Subflow. Once saved, a template is available to select when composing broadcasts, responding in conversations, and building workflows.

## Before you start

There are no prerequisites for creating a message template.

## Create an SMS message template

SMS message templates are text-based and can include personalized patient information. The steps below walk you through creating one.

1. **Navigate to** **Messaging** in the sidebar. Under **Campaigns**, select **Message Templates**.

2. **Click** **New Message Template** in the top right corner.

3. **Enter a name** in the **Title** field. The title helps you identify the template when selecting it later.

4. **Write your message** in the **Content** field. The content area has a 1,530-character limit.

   - To personalize the message with patient information, click **Personalize** in the content toolbar. A **Contact Properties** list appears. Select a property to insert it into the message.
   - To start from an existing template instead of writing from scratch, click **Templates** in the content toolbar and select a template to load its content.
   - To attach a file or resource, click **Attach** in the content toolbar.

   > **Note:** The inserted contact properties appear as chips in the message content. The team is finalizing the term to use for these items across the documentation — this guide uses "contact property" to match the label shown in the UI. Update this term once the team reaches a decision.

5. **Add tags** in the **Tags** field (optional). Tags make the template easier to find when filtering or searching the templates list.

6. **Click** **Save**. The template appears in your list of message templates and is now available to select in broadcasts, conversations, and workflows.
